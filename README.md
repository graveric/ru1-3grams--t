# ru1-3grams--t

Улучшенная статистика альф, биграмм и триграмм для русского языка.

## Описание

Этот проект предоставляет инструменты и данные для анализа статистики n-грамм в русском языке:
- **Альфа** (unigrams) - одиночные символы
- **Биграммы** (bigrams) - последовательности из двух символов
- **Триграммы** (trigrams) - последовательности из трех символов

## О проекте

Проект ru1-3grams--t содержит улучшенную статистическую информацию о частоте встречаемости различных комбинаций символов в русском языке. Эти данные могут быть полезны для:
- Лингвистических исследований
- Криптоанализа
- Машинного обучения и NLP задач
- Разработки клавиатурных раскладок
- Анализа текстов

# ru1-3grams--t (English)

Improved statistics of alphas, bigrams, and trigrams for the Russian language.

## Description

This project provides tools and data for analyzing n-gram statistics in Russian language:
- **Alphas** (unigrams) - single characters
- **Bigrams** - sequences of two characters
- **Trigrams** - sequences of three characters

## About

The ru1-3grams--t project contains improved statistical information about the frequency of various character combinations in the Russian language. This data can be useful for:
- Linguistic research
- Cryptanalysis
- Machine learning and NLP tasks
- Keyboard layout development
- Text analysis
